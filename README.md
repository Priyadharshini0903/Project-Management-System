📋 Project Management System (VB.NET)
📌 Overview

The Project Management System is a desktop application developed using VB.NET to help teams organize and track project tasks efficiently.
It allows users to create projects, assign tasks, and manage task progress through different workflow stages using a drag-and-drop task board.

The system provides a simple Kanban-style interface where tasks move between stages such as To-Do, Doing, and Done, making project tracking easier and more visual.

🎯 Key Features

📂 Project Management
Create and manage multiple projects.

✅ Task Creation & Assignment
Add tasks with details like:

• Assigned user

• Priority level

• Due date

🖱 Drag and Drop Task Board
Tasks can be moved between stages:

• To-Do

• Doing

• Done

👤 User-Based Task Tracking
Tasks display the assigned user for easy responsibility tracking.

⚡ Priority Management
Tasks are categorized based on priority:

• High

• Medium

• Low

📊 Dashboard View
Provides an overview of project tasks and their progress.

✏ Task Editing & Deletion
Users can update or remove tasks when needed.

🔄 Real-Time Refresh
Refresh option to reload and view updated tasks instantly.

🖥 Application Interface

• The application interface includes:
